Distraction Garden Game
Game edukasi tentang menjaga fokus di dunia digital!

A. Deskrisi singkat Aplikasi

  Distraction Garden adalah game edukasi yang dikembangkan menggunakan PyGame, menyimulasikan perjuangan sehari-hari untuk mempertahankan Fokus di tengah badai interupsi digital. Anda berperan sebagai Digital Gardener, yang bertugas melindungi Tanaman dari serangan Bug Distraksi. Hal ini diibaratkan sebagai fokus pengguna yang menjadi simbol energi mental dan produktivitas  

B.Fitur Utama
1. 3 Level dengan kesulitan meningkat
2. 3 Jenis Bug Distraksi dengan perilaku unik
3. Sistem Fokus Dinamis yang menurun seiring waktu
4. Quiz Mini-game untuk melawan popupbug
5. Efek Visual Partikel yang memukau
6. Audio Feedback lengkap
7. UI Responsif dengan animasi

B. Cara Main
1. Install Python
2. Intall Pygame
3. Run main.py
4. Mainkan setiaplevel gamenya

C. Kontrol Permainan
1. Arrow Keys / WASD: Gerakkan karakter
2. Space: untuk menghancurkan bug
3. ESC: Kembali ke menu
4. Arrow Keys/TouchPad: Navigasi menu

D. Struktur file
1. data/: berisi pertanyaan untuk setiap level
2. entities/: berisi karakter dan musuh
3. scenes/: berisi tampilan halaman menu,level dan pertanyaan
4. sounds/: berisi sound efek dan music background game
5. main.py: penanggung jawab loop utama game
